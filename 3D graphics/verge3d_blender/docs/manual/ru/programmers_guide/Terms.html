Словарь терминов компьютерной графики и веб-программирования

Ниже приведён список русскоязычных терминов и определений, используемых в компьютерной графике и веб-программировании.

2Д
относящийся к двумерному пространству. Модель мира, в которой имеется всего два измерения: длина, ширина или иначе — плоская модель мира.
3Д
относящийся к трёхмерному пространству. Модель мира, в которой имеется три измерения: длина, ширина, высота (глубина) или иначе — объёмная модель мира.
3Д-веб
набор веб-технологий и сопутствующего программного обеспечения для создания и отображения трёхмерной графики в интернете, в том числе с использованием устройств виртуальной и дополненной реальности. Включает в себя: ВебГЛ, ВебИксЭр, ГЛТФ, «Вердж3Д» и др.
3дс Макс
коммерческий пакет для создания трёхмерной графики, разрабатываемый компанией «Автодеск».
Автодеск
крупнейшая компания-разработчик инженерного и графического программного обеспечения из США.
Автокад
профессиональная система автоматизированного проектирования (САПР) и черчения. Разрабатывается компанией «Автодеск», США.
АПИ
то же что и «программный интерфейс», способ взаимодействия программы с другой программой или библиотекой.
Арнольд
профессиональный движок для рендеринга (рендер), использующий трассировку лучей для получения фотореалистичных изображений. Разрабатывается компанией «Автодеск», входит в состав ряда продуктов компании: «3дс Макс», «Майя». Также имеется отдельная платная версия.
БААА
быстрый аппроксимирующий анти-алиасинг. Основанный на постпроцессинге алгоритм сглаживания 3Д-визуализации, разработанный компанией «Нвидиа».
Блендер
пакет трёхмерного моделирования, анимации и рендеринга с открытым исходным кодом. Распространяется свободно.
Буллет
программная библиотека для проведения физических симуляций (физический движок) с открытым исходным кодом. Основные сферы применения — разработка компьютерных игр и приложений виртуальной реальности.
Вавилон, Вавилон-Джей-Эс
открытый движок для создания трёхмерных визуализаций в интернете, разрабатывается компанией «Майкрософт», США.
Веб Аудио
технология работы со звуком с использованием веб-браузеров. Используется для игр, интерактивных визуализаций, виртуальной реальности, а также для профессионального ПО для обработки звука. Поддерживается во всех современных веб-браузерах.
ВебГЛ
стандарт рендеринга трёхмерной графики в интернете. Поддерживается во всех современных веб-браузерах.
ВебГПУ
экспериментальный стандарт рендеринга трёхмерной графики в интернете, идущий на смену «ВебГЛ».
ВебИксЭр
технология взаимодействия с устройствами виртуальной и дополненной реальности из веб-браузеров.
Вебфлоу
визуальный редактор, использующийся для создания веб-страниц без программирования и вёрстки. Разрабатывается одноименной компанией «Вебфлоу», США.
Вердж3Д
программное обеспечение для создания трёхмерных веб-приложений. Используется для создания интернет-магазинов, конфигураторов товаров, 3Д-презентаций, обучающих курсов, научных приложений, приложений виртуальной и дополненной реальности, браузерных игр и др.
ВИЗИВИГ
техника создания контента, при которой разрабатываемый продукт выглядит максимально приближенным к конечному результату. Термин обычно используется в контексте программного обеспечения для создания документов, презентаций и веб-страниц. Примеры ВИЗИВИГ-редакторов: «Майкрософт Офис», «Либреофис», «Вебфлоу».
Вирей
профессиональный движок для рендеринга (рендер), использующий трассировку лучей для получения фотореалистичных изображений. Разрабатывается компанией «Хаос Груп», Болгария.
Вордпресс
открытая система управления контентом, написанная на языке ПХП. Используется для создания сайтов любой сложности, на сегодняшний день является крупнейшей в мире по числу установок.
Вукоммерц
система электронной коммерции, выполненная в виде плагина для системы управления контентом «Вордпресс». Является наиболее популярным решением для создания интернет-магазинов.
Вьюпорт 2.0
движок для рендеринга (рендер), используемый в программе «Майя» для отрисовки содержимого вьюпорта. Обладает высокой производительностью, позволяя получать результат рендеринга в реальном времени.
ГИМП
cвободно-распространяемая программа для редактирования изображений.
ГЛТФ
открытый формат файлов, используемый для хранения и передачи 3Д-графики. Файлы в этом формате могут содержать геометрию, положение объектов, структуру сцен, анимацию, данные камеры, источники света, материалы, текстуры и другую информацию. Формат создан с расчётом на применение в интернете.
Джейсон
простой формат для хранения и обмена структурированными данными. Основные сферы использования: веб-программирование, хранение настроек приложений, 3Д-веб (в частности, формат ГЛТФ был разработан на базе «Джейсона»).
Директ3Д
составная часть «ДиректИкс», программный интерфейс (АПИ) для создания трёхмерной графики для ОС «Виндоус».
ДиректИкс
набор программных интерфейсов для создания компьютерных игр и других мультимедийных приложений для операционной системы «Виндоус».
Иви
открытый движок для рендеринга (рендер), используемый в программе «Блендер» для отрисовки содержимого вьюпорта. Обладает высокой производительностью, позволяя получать результат рендеринга в реальном времени.
Иллюстратор
редактор векторной графики, разрабатываемый компанией «Адоби», США.
Кубомапа
жаргонное название кубической текстуры (кубической карты), то есть текстуры, используемой для наложения отражений на трёхмерные объекты.
КОЛЛАДА
открытый формат хранения и обмена трёхмерными данными. В настоящее время вытесняется форматами ГЛТФ и ЮСД.
Кордова
открытый инструмент для преобразования веб-приложений в кроссплатформенные мобильные приложения, работающие на платформах «Айос» и «Андроид».
КУ́ДА
архитектура графических процессоров и программный интерфейс для высокопроизводительных параллельных вычислений, разработанный компанией «Нвидиа». На текущий момент поддерживается исключительно на оборудовании компании «Нвидиа».
ЛЗМА
высокоэффективный алгоритм сжатия (компрессии) данных, а также формат файлов, сжатых с помощью этого алгоритма. Разработан российским программистом Игорем Павловым. В частности, данный алгоритм используется в движке «Вердж3Д» для сжатия геометрии и метаданных трёхмерных сцен, с целью ускорения загрузки приложений.
Майя
пакет трёхмерного моделирования, анимации и рендеринга от компании «Автодеск».
Меш
описание геометрии объекта в виде вершин, рёбер и граней. Зачастую является синонимом слов «геометрический объект» или «полигональный объект».
МСАА
мультисемплинговый анти-алиасинг. Техника сглаживания 3Д-визуализации, использующая возможности графического оборудования по рендерингу геометрии на суб-пиксельном уровне.
Ноуд, Ноуд-Джей-Эс
платформа для запуска языка программирования «ЯваСкрипт» за пределами веб-браузера. Позволяет создавать десктопные и серверные приложения с использованием данного языка.
НПМ
пакетный менеджер и коллекция готовых пакетов для разработчиков приложений на платформе «Ноуд-Джей-Эс».
ОпенГЛ
стандарт и программный интерфейс (АПИ) для создания приложений, работающих с трёхмерной графикой.
ОпенСЛ
открытый и кроссплатформенный программный интерфейс для высокопроизводительных параллельных вычислений. Является открытым аналогом технологии «КУДА».
ПБР
физически корректный рендеринг, набор алгоритмов для симуляции особенностей распространения света. Позволяет добиться высокой степени реализма изображения.
ПДФ
популярный формат распространения электронных документов, разработанный компанией «Адоби». Файлы данного формата имеют расширение .pdf.
Питон
многофункциональный язык программирования. Основные сферы применения: веб-разработка, написание встроенных скриптов для ряда приложений, научные и инженерные расчёты, анализ данных.
Полигон
треугольник, четырёхугольник или многоугольник, используемый для описания геометрии объектов (мешей).
Программный интерфейс
то же что и «АПИ», способ взаимодействия программы с другой программой или библиотекой.
ПХП
популярный серверный язык программирования, используемый для разработки веб-сайтов.
Растеризация
процесс преобразования векторной или полигональной графики в растровое (состоящее из пикселей) изображение.
Рейтрейсинг
см. «трассировка лучей».
Рендер, рендерер
программное обеспечение для осуществления рендеринга, т.е. отрисовки трёхмерных моделей.
Рендеринг
процесс визуализации трёхмерной модели с помощью специализированной программы-рендера.
РЕСТ, РЕСТ АПИ
набор правил взаимодействия программы с сервером. Чаще всего подразумевает набор специализированных УРЛ-ссылок, формирующий АПИ сервера, а также формат передачи данных между клиентом и сервером на основе «Джейсон».
Сабстанс Пейнтер
программного обеспечение для текстурирования трёхмерных моделей. Разрабатывается компанией «Адоби».
Сайклз
движок для рендеринга (рендер), использующий трассировку лучей для получения фотореалистичных изображений. Распространяется свободно, входит в состав пакета 3Д-моделирования «Блендер».
Синема 4Д
коммерческий пакет для создания трёхмерной графики, разрабатываемый компанией «Максон», Германия.
Сканлайн
устаревший движок для рендеринга, поставляемый в составе пакета моделирования «3дс Макс». В настоящее время заменяется рендером «Арнольд».
СКОРМ
стандарт систем электронного обучения, а также формат обучающих курсов, совместимых с данными системами.
Спекуляр, спекулярное отражение
зеркальное отражение, блик. Специфичный термин, используемый в компьютерной графике для обозначения отражения, противоположного по смыслу диффузному.
Трассировка лучей
или рейтрейсинг, метод визуализации моделей (рендеринг), основанный на симуляции взаимодействия поверхности с отдельными лучами света. Позволяет получить реалистичное изображение с качеством приближенным к фотографии (фотореализм). Недостаток — высокая вычислительная сложность алгоритмов, приводящая к низкой производительности рендеров, использующих трассировку лучей.
Три-Джей-Эс
открытый движок для создания трёхмерных визуализаций в интернете.
УРИ
Унифицированный ресурсный идентификатор — строка идентифицирующая некоторый ресурс, чаще всего представляет собой интернет-ссылку (УРЛ).
УРЛ
Унифицированный ресурсный локатор — ссылка на какой-либо ресурс в интернете.
Фотошоп
программа для редактирования изображений от компании «Адоби».
Фрустум каллинг
техника оптимизации рендеринга. Заключается в отсечении геометрии не попадающей в объём камеры.
ХДР
имеющий высокий динамический диапазон. В контексте компьютерной графики обозначает текстуру или рендер с увеличенным диапазоном цветопередачи (больше 8-бит на каждый канал цвета).
ХТМЛ
язык разметки гипертекста. Используется для определения содержимого и внешнего вида веб-страниц.
ЦСВ
простой формат для хранения табличных данных. Строки таблицы представляют собой строчки текстового файла, колонки разбиваются с помощью запятых (реже других символов).
ЦСС
язык каскадных таблиц стилей. Используется совместно с ХТМЛ для задания внешнего вида веб-страниц.
шейдер
короткая программа выполняющаяся на графическом процессоре. Шейдеры чаще всего пишутся с использованием специализированных языков программирования, таких как ГЛСЛ (язык шейдеров ОпенГЛ), ХЛСЛ (язык шейдеров ДиректИкс), ОСЛ (открытый язык для рендеров, использующих рейтрейсинг) и др.
ШПЛ
шутер от первого лица, жанр компьютерных игр в которых управление камерой производится от первого лица. Термин применяется также и вне контекста компьютерных игр (в частности, в 3Д-веб-приложениях) для обозначения аналогичного способа управления камерой.
Электрон
открытое программное обеспечение для преобразования веб-приложений в кросс-платформенные десктопные приложения.
Эмбиент окклюжн
техника улучшающая реалистичность рендеринга посредством дополнительного затенения от непрямых источников. Степень этого затенения определяется тем, насколько видима рисуемая часть объекта, если смотреть не из камеры, а со стороны источников освещения.
Юнити
игровой движок разрабатываемый компанией «Юнити Технолоджис», США. Основное применение — создание игр для мобильных устройств.
ЮСД, ЮСДЗ
формат файлов, используемый для хранения 3Д-графики. Файлы в этом формате могут содержать геометрию, положение объектов, структуру сцен, анимацию, данные камеры, источники света, материалы, текстуры и другую информацию. Cоздан компанией «Пиксар», среди прочего используется на устройствах «Эпл» («Айфон», «Айпад») для отображения моделей в режиме дополненной реальности. Формат ЮСДЗ отличается от базового варианта ЮСД компоновкой всех содержащихся данных в едином файле, упакованном с помощью алгоритма компрессии «ЗИП» (отсюда и последняя буква в названии).
