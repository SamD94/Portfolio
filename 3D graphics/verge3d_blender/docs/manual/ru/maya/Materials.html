Шейдеры поверхностей

#

Standard Surface

Шейдер Standard Surface был представлен в «Майе» в 2020 году. Он основан на aiStandardSurface, поэтому очень на него похож.

Поскольку он лучше поддерживается в рендере «Вьюпорт» 2.0 и не требует установки плагина Arnold, мы рекомендуем использовать его при создании контента «Вердж3Д» в «Майе» 2020 или выше.

#

Lambert

Базовая, но очень эффективная нода, которыая может быть использована для представления материалов без спекулярных отражений.

В «Вердж3Д» поддерживаются следующие настройки Lambert:

  • Color
  • Transparency
  • Diffuse
  • Bump/Normal Mapping

#

Blinn, Phong, Phong E

В «Вердж3Д» эти материалы будут приблизительно соответствовать модели шейдинга PBR, поэтому они могут выглядеть по-другому по сравнению с вьюпортом «Майи».

#

Surface Shader

Используйте эту шейдерную ноду для создания полностью настраиваемых материалов. По сути, он просто передает параметр Out Color в рендер как есть.

Arnold Nodes

#

aiStandardSurface

Этот шейдер поставляется с «Арнольдом» для «Майи» (MtoA) и может помочь реализовать многие реальные материалы (такие как металл, пластик, стекло, дерево, текстиль и т.д.) с помощью модели шейдинга основанной на физике.

В «Вердж3Д» поддерживаются следующие настройки aiStandardSurface:

Base Panel

  • Weight or (Base вход в Hypershade)
  • Color or (Base Color вход в Hypershade)
  • Metalness

Specular Panel

  • Weight (Specular вход в Hypershade)
  • Color (Specular Color вход в Hypershade)
  • Roughness (Specular Roughness вход в Hypershade)
  • IOR (Specular IOR вход в Hypershade)

Transmission Panel

  • Weight (Transmission вход в Hypershade)
  • Color (Transmission Color вход в Hypershade)

Coat Panel

  • Weight (Coat вход в Hypershade)
  • Roughness (Coat Roughness вход в Hypershade)
  • Normal (Coat Normal вход в Hypershade)

Sheen Panel

  • Weight (Sheen вход в Hypershade)
  • Color (Sheen Color вход в Hypershade)
  • Roughness (Sheen Roughness вход в Hypershade)

Emission Panel

  • Weight (Emission вход в Hypershade)
  • Color (Emission Color вход в Hypershade)

Geometry Panel

  • Thin Walled
  • Opacity
  • Bump Mapping (Normal Camera вход в Hypershade)

Дополнительные настройки «Вердж3Д»

Вы можете настроить материалы aiStandardSurface с помощью следующих параметров, расположенных на панели Verge3D:

Alpha Mode
режимы прозрачности: Auto, Opaque, Blend или Mask. См. руководство по прозрачности для дополнительной информации.
Two Sided
делает ваш материал двухсторонним. По умолчанию «Вердж3Д» рендерит только лицевую сторону материалов, включите эту опцию, чтобы отрендерить и обратную сторону.
Depth Write
рендерит глубину материала в буфере глубины. Используйте эту опцию для прозрачных материалов, чтобы сделать их рендеринг аддитивным.
Depth Test
выполняет тест глубины при рендеринге материала. Отключение теста глубины полезно при рендеринге объектов внутри полупрозрачных объектов, чтобы предотвратить их вытеснение покрывающими объектами.
Dithering
уменьшение артефактов-полос с помощью пиксельного дизеринга.
glTF 2.0 Compatible
делает материал совместимым с glTF 2.0. Пройдите по ссылке для получения дополнительной информации.

Производительность

Standard Surface (или aiStandardSurface), являясь единственным выбором для создания красивых и реалистичных рендеров, тем не менее, требует больше вычислительной мощности от GPU и замедляет загрузку приложений. Это особенно актуально для сцен с большим количеством таких материалов.

Для повышения производительности вы можете настроить разрешение текстур или режим IBL Environment.
