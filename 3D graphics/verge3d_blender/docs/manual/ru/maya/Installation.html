Предварительные условия

«Майя»

«Вердж3Д» работает поверх графического пакета «Майя» 2019-2023, который необходимо установить первым. Мы рекомендуем использовать последнюю версию «Майи» (на момент написания это версия 2023) для достижения наилучших результатов.

Операционная система

Мы рекомендуем использовать «Вердж3Д» на «Виндоус» 10/11, «МакОС» 12/13 или последней версии «РедХат»/«ЦентОС»/«Федора» «Линукс». Убедитесь, что у вас установлены последние версии видеодрайверов. Для установки не требуются права администратора.

Веб-браузер

«Вердж3Д» может работать с любыми из следующих браузеров: «Хром», «Файрфокс», «Опера», или «Эдж». Однако мы рекомендуем использовать браузер «Хром», поскольку он является быстрым, совместимым со стандартами и предлагающим удобные инструменты для веб-разработки. Возможно, вы захотите сделать его браузером по умолчанию для более удобного предварительного просмотра.

Установка «Вердж3Д» в «Виндоус»

Скачайте пакет «Вердж3Д» и дважды щелкните на файле .exe, чтобы запустить программу установки:

Проследуйте по ссылке чтобы ознакомиться с лицензионным соглашением. В программе установки лицензионное соглашение представлено на английском языке. Русская версия расположена по ссылке.

Выберете компоненты для установки:

Выберите папку для установки (либо оставьте имеющийся вариант). Убедитесь, что у вас достаточно разрешений на чтение и запись в данную папку.

Дождитесь завершения копирования файлов:

По окончании установки вы должны увидеть следующий экран:

Нажмите кнопку Finish и запустите «Майю». Если она был запущен во время установки, закройте и откройте её заново. Теперь перейдите к настройкам Plug-in Manager...

...и активируйте плагин verge3d.py, включив флажки Loaded и Auto load.

Дополнительно вы можете активировать плагин gltfExport.py, если хотите экспортировать сцены «Вердж3Д» через опцию меню File → Export All.... Включение gltfExport.py добавляет опцию типа файла Verge3D GLTF для диалогового окна Export All.

Теперь вы должны увидеть меню Verge3D в строке главного меню.

Установка завершена!

Также теперь у вас есть возможность запускать менеджер проектов «Вердж3Д» независимо от «Майи», нажав на значок на рабочем столе или используя меню «Пуск».

Установка «Вердж3Д» в «МакОС»

Скачайте архив «Вердж3Д» для «МакОС» (имеет расширение .xz) и распакуйте его в выбранный вами каталог. Убедитесь, что у вас достаточно прав для чтения/записи в этой папке. Установка в системный каталог не рекомендуется.

Если вы используете «Майю» версии до 2022 года, вам необходимо выполнить дополнительные шаги для установки Python (который входит в состав Command Line Tools). Если вы используете «Майю» 2022 или новее, перейдите к шагу с вызовом Plug-in Manager.

Для установки Command Line Tools откройте терминал «МакОС» и выполните команду:

xcode-select --install

Если инструменты не установлены в вашей системе, сценарий вызовет следующее диалоговое окно установки:

Дождитесь завершения установки, прежде чем приступать к дальнейшим действиям.

Теперь откройте «Майю» и перейдите к настройкам Plug-in Manager...

Нажмите на кнопку Browse

затем перейдите в директорию Path_To_Verge3D_Installation/maya_plugin/setup и откройте скрипт install.py.

В более новых версиях «Майи» может появиться следующее диалоговое окно - нажмите Allow.

Перезапустите «Майю», снова откройте настройки Plug-in Manager и активируйте плагин verge3d.py, включив флажки Loaded и Auto load.

Если хотите экспортировать сцены «Вердж3Д» через опцию меню File → Export All... вам необходимо активировать плагин gltfExport.py. Включение gltfExport.py добавляет опцию типа файла Verge3D GLTF для диалогового окна Export All.

Теперь вы должны увидеть меню Verge3D в строке главного меню.

Установка завершена!

#

Настройка плагина вручную

Иногда меню Verge3D не отображается. Это может произойти по нескольким причинам:

  • Вы отключили опцию Auto-Configure в программе установки.
  • Вы установили и запустили неподдерживаемую версию «Майи».
  • Вы сменили пользователя «Виндоус».

В этом случае вы можете использовать следующую процедуру для настройки пути к плагину вручную.

Найдите файл с именем Maya.env, который обычно находится в следующих папках:

  • «Виндоус» — \Users\user\Documents\maya\VERSION
  • «МакОС» — $HOME/Library/Preferences/Autodesk/maya/VERSION
  • «Линукс» — $HOME/maya/VERSION

Отредактируйте этот файл, чтобы указать переменную окружения MAYA_PLUGIN_PATH следующим образом:

MAYA_PLUG_IN_PATH=\Users\user\verge3d_maya\maya_plugin

Сохраните файл и откройте «Майю». Если она была запущена, не забудьте закрыть и открыть ее снова. Найдите плагин Verge3D в диалоге настроек Plug-In Manager. Активируйте его, нажав на флажки Loaded и Auto load. Теперь вы должны увидеть меню Verge3D в строке главного меню.
