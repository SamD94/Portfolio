电子商务插件 | E-Commerce Plugin

此类目拼图通过Verge3D 插件 功能创建，生成了可用于实现WooCommerce 3D产品视图和配置器的拼图，也可用于创建独立的订单或咨询表单。

内容

  • 拼图参考

    • connect to WooCommerce(连接到WooCommerce)
    • product info(产品信息)
    • product attribute(产品属性)
    • place order(下单)
    • place order +
    • order item (sku, qty)

paypal create order(创建Paypal订单)

下PayPal订单。此拼图是最简单的电子商务解决方案，因为它无需任何服务器端配置。要使用他，您需要由已通过验证的PayPal企业账号。

client ID
PayPal企业账号客户ID，请在 此页 获取。其提供了用于测试目的沙盒凭证，如果要用于真实销售，您需要执行 额外的步骤 来接收实时凭证。
title
产品标题。
item price
产品单价。
currency
货币。
quantity
产品数量。
on success do
此插槽中的拼图将在交易成功后执行。
on error do
此插槽中的拼图将在遇到错误时执行（例如资金不足等情况）。

连接到WooCommerce系统，开始接收产品信息。

例如，请参考资产商店中的以下演示 — Drink Shop.

详情见此教程： 英文版 ，中文版。

#

product info(产品信息)

返回一个 dictionary(字典) ，其中包含由 connect to WooCommerce 拼图检索的产品信息。这个字典（JavaScript对象）中包含以下字段。

  • price - 客户选择的产品选项的价格，例如：12。
  • quantity - 客户选择的产品项目的数量，例如：2。
  • sku - 客户选择的产品选项的库存单位。
  • name - 产品的名称，因为它将在商店的购物车中看到，例如 "Spinner - Yellow, Large" 。
  • attributes - 一个包含客户所选产品选项的属性名称/值对的字典，例如：{color: "Yellow", size: "Large"}.
  • weight - 客户选择的产品选项的实际产品重量。
  • length, width, height - 客户选择的产品选项的实际产品尺寸。
  • type - 在WooCommerce管理面板中指定的以下产品类型之一： "variable", "simple", "group" 或 "external" 。
  • children - 只对 "group" 产品有效。包括每个子产品的信息。
  • status - 服务器响应： "ok" 或 "error" 。

#

product attribute(产品属性)

返回客户选择的产品选项属性值，例如： "Yellow" 。

详情见此教程： 英文版 ，中文版。

#

place order(下单)

编写订单并将其提交到指定的 URL。默认情况下，订购表单会被提交到 sandbox ordering page。

例如，请参考资产商店中的以下演示 — Ring.

Verge3D带有一个免费的Wordpress插件，用于处理由这个拼图提交的请求。收到此类请求后，此Wordpress插件会生产带有扩展表单的页面，并补充联系人、评论、屏幕截图等字段。填好的表单最后由客户提交，并在Wordpress管理界面创建一个新的订单。

send to
WordPress 插件提供的订单表单的 URL。请参考本手册的 Wordpress插件 章节，了解设置方法。
content
订单内容（以人类可读的形式呈现）。
item price
产品项目价格。
quantity
产品的数量。
make screenshot
生成应用屏幕截图并将其附加到订单中。请确保在 configure application(配置应用程序) 拼图中启用屏幕截图，否则屏幕截图可能会显示为纯黑色。

由您（或销售经理）来验证订单内容和价格的正确性。尽快可能性很小，但黑客可能会通过篡改拼图提交的数据的方式来以较低的价格购买产品。为防止此类事件，您需要使用 place order + 拼图依赖服务器上分配的产品信息来验证订单。

#

place order +

编写订单并将其提交到指定的 URL。此拼图工作方式类似于 place order ，但有更多特性，安全性也更高，并且允许自定义。

例如，请参考资产商店中的以下演示 — Recliner.

send to
WordPress 插件提供的订单表单的 URL。有关设置信息，请参阅 此处指南 。
item(s)
订单项目或订单项目列表。参阅 下文 了解详情。
attachment(s)
订单附件或订单附件列表。附件是一个 Data URI link ，用来表示屏幕截图、图片、声音，视频，PDF，文本或glTF模型。
