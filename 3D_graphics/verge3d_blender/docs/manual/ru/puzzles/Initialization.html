Пазлы инициализации (Initialization)

Эти пазлы находятся во вкладке init и интерпретируются перед инициализацией приложения Вердж3Д.

Содержание

  • Перечень пазлов
    • configure application
    • setup preloader + percentage
    • load font

#

Перечень пазлов

#

configure application

Позволяет задать параметры инициализации приложения, включая некоторые параметры создания контекста ВебГЛ.

compressed assets
заставляет приложение загружать сжатую сцену в формате .xz вместо обычного .gltf (см. Asset Compression для большей информации).
default fullscreen button
включает стандартную кнопку полноэкранного режима, расположенную в правом верхнем углу.
transparent background
делает фон прозрачным, чтобы основные части веб-страницы были видны через канвас ВебГЛ. Включите этот параметр, чтобы встроить 3Д-контент на веб-страницу или использовать функцию дополненной реальности.
enable screenshots
позволяет снимать скриншоты окна рендеринга (устанавливает свойство preserveDrawingBuffer контекста ВебГЛ в true).
fade annotations
делает аннотации тусклыми, когда они блокируются объектами сцены.

#

setup preloader + percentage

Удаляет прелоадер по умолчанию и передает обратные вызовы событий для обработки событий прогресса прелоадера.

В следующем примере пользовательский прелоадер представлен с контейнером ХТМЛ-элементом (например, <div>) с идентификатором "preloader". Внутри него находится другой элемент с идентификатором "loading progress" (например, <span>), в котором выводится процент загрузки с помощью пазла percentage. Кроме того, ширина третьего элемента (например, <div>) динамически изменяется для графической визуализации прогресса загрузки.

#

load font

Загружает шрифт из указанного УРЛ и делает его доступным через ЦСС-правило font-family.
