Установка / Блендер

  • Необходимые условия
  • Загрузка «Вердж3Д»
  • Установка в «Виндоус»
  • Установка в «МакОС» и «Линукс»

#

Необходимые условия

«Вердж3Д» может работать поверх «Блендера», пакета 3Д-моделирования и анимации с открытым исходным кодом, который необходимо установить заранее (поддерживаются версии 2.83+). Вы можете бесплатно загрузить «Блендер» с официального сайта.

#

Загрузка «Вердж3Д»

Загрузите «Вердж3Д для Блендера» для своей операционной системы (установщик .exe для «Виндоус» или архив .xz для «МакОС»/«Линукс»).

#

Установка в «Виндоус»

Дважды щелкните файл .exe, чтобы запустить установщик:

Проследуйте по ссылке чтобы ознакомиться с лицензионным соглашением. В программе установки лицензионное соглашение представлено на английском языке. Русская версия расположена по ссылке.

Выберете компоненты для установки:

Выберите папку для установки (либо оставьте имеющийся вариант). Убедитесь, что у вас достаточно разрешений на чтение и запись в данную папку.

Дождитесь завершения копирования файлов:

По окончании установки вы должны увидеть следующий экран:

Нажмите кнопку Finish и запустите «Блендер». Если он был запущен во время установки, закройте и откройте его заново.

Откройте окно Preferences и перейдите на вкладку Add-ons. Найдите и включите аддон «Вердж3Д», после чего закройте окно.

Установка завершена!

Вы также можете запускать менеджер приложений без необходимости запуска «Блендера». Для этого щелкните на соответствующем значке на рабочем столе или используйте меню «Пуск»:

#

Установка в «МакОС» и «Линукс»

Распакуйте архив с программой «Вердж3Д» (имеет расширение XZ) в каталог по вашему выбору. Убедитесь, что у вас достаточно разрешений на чтение и запись в эту папку. Распаковка в системный каталог не рекомендуется.

В «Блендере» вызовите окно Preferences, выбрав соответствующую опцию в меню Edit.

На вкладке File Paths найдите поле Scripts и укажите в нем папку, в которой установлен «Вердж3Д». Закройте окно Preferences и перезапустите «Блендер».

Снова откройте окно Preferences и перейдите на вкладку Add-ons. Найдите и включите аддон «Вердж3Д», после чего закройте окно.

Установка завершена!
